# M8:安全加固与个人中心 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现改自己密码(校验旧密码+踢会话)、管理员重置他人密码(默认 123456+踢会话)、个人资料编辑(昵称/手机/邮箱/头像URL,查重)、个人中心页;并把 reset 接口纳入动态RBAC 映射防越权。

**Architecture:** 后端在现有 `AuthService`(自助:profile/password)与 `SysUserService`(管理员:resetPassword)上加方法,复用已注入的 `userMapper`/`passwordEncoder`(两者均 Lombok `@RequiredArgsConstructor`,不改构造器)。`StpUtil.logout(userId)` 踢会话。reset 接口经种子写入 `sys_menu` 的 api_url→perm 映射,由 `ApiPermPublisher` 启动发布、网关动态查表鉴权。前端加个人中心页 + 用户管理重置按钮。

**Tech Stack:** 后端沿用(Sa-Token + MyBatis-Plus + BCrypt + Redis 动态RBAC);前端 Vue3 + Element Plus。

> 路径基准:仓库根 `auth-wjb/`。Maven 走 `build.ps1`(固定 JDK21);指定测试加 `"-Dsurefire.failIfNoSpecifiedTests=false"`。当前分支 `master`,工作区干净。

- [ ] **Step 2: 在类末尾 `currentUserMenus()` 方法之后(最后一个 `}` 之前)加入三个方法**

```java
    /** 当前用户资料 */
    public ProfileVO getProfile() {
        SysUser user = userMapper.selectById(UserContext.getUserId());
        if (user == null) {
            throw new ServiceException("用户不存在");
        }
        ProfileVO vo = new ProfileVO();
        vo.setUsername(user.getUsername());
        vo.setNickname(user.getNickname());
        vo.setPhone(user.getPhone());
        vo.setEmail(user.getEmail());
        vo.setAvatar(user.getAvatar());
        return vo;
    }

    /** 改自己资料(昵称/手机/邮箱/头像);手机、邮箱查重(排除自己) */
    public void updateProfile(ProfileUpdateRequest req) {
        Long userId = UserContext.getUserId();
        SysUser user = userMapper.selectById(userId);
        if (user == null) {
            throw new ServiceException("用户不存在");
        }
        if (StringUtils.hasText(req.getPhone())) {
            Long c = userMapper.selectCount(new LambdaQueryWrapper<SysUser>()
                    .eq(SysUser::getPhone, req.getPhone())
                    .ne(SysUser::getId, userId));
            if (c != null && c > 0) {
                throw new ServiceException("手机号已被占用");
            }
        }
        if (StringUtils.hasText(req.getEmail())) {
            Long c = userMapper.selectCount(new LambdaQueryWrapper<SysUser>()
                    .eq(SysUser::getEmail, req.getEmail())
                    .ne(SysUser::getId, userId));
            if (c != null && c > 0) {
                throw new ServiceException("邮箱已被占用");
            }
        }
        user.setNickname(req.getNickname());
        user.setPhone(req.getPhone());
        user.setEmail(req.getEmail());
        user.setAvatar(req.getAvatar());
        userMapper.updateById(user);
    }

    /** 改自己密码:校验旧密码 → 存新 → 踢当前会话强制重登 */
    public void changePassword(ChangePasswordRequest req) {
        Long userId = UserContext.getUserId();
        SysUser user = userMapper.selectById(userId);
        if (user == null) {
            throw new ServiceException("用户不存在");
        }
        if (user.getPassword() == null || !passwordEncoder.matches(req.getOldPassword(), user.getPassword())) {
            throw new ServiceException("原密码错误");
        }
        user.setPassword(passwordEncoder.encode(req.getNewPassword()));
        userMapper.updateById(user);
        StpUtil.logout(userId);
    }
```
